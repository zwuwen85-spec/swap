# 校园闲置物品交换平台 - 开发流程与计划

## 一、开发流程

### 1.1 敏捷开发流程

```
需求分析 → 技术设计 → 开发实现 → 单元测试 → 代码评审 → 集成测试 → 部署上线
    ↑                                                                   ↓
    └────────────────────────── 迭代反馈 ←───────────────────────────────┘
```

---

### 1.2 Git工作流

```
main（生产环境）
  ↑
  └── release（预发布环境）
         ↑
         └── develop（开发环境）
                ↑
                ├── feature/xxx（功能分支）
                ├── bugfix/xxx（修复分支）
                └── hotfix/xxx（紧急修复）
```

**分支规范：**
- `main`: 生产环境，只接受merge请求
- `develop`: 开发环境，日常开发分支
- `feature/*`: 新功能分支
- `bugfix/*`: Bug修复分支
- `hotfix/*`: 紧急修复分支

---

### 1.3 提交规范

```
<type>(<scope>): <subject>

<body>

<footer>
```

**type类型：**
- `feat`: 新功能
- `fix`: Bug修复
- `docs`: 文档更新
- `style`: 代码格式（不影响功能）
- `refactor`: 重构
- `test`: 测试相关
- `chore`: 构建/工具相关

**示例：**
```
feat(user): 添加用户注册功能

实现用户注册接口，包括：
- 参数验证
- 密码加密
- JWT Token生成

Closes #123
```

## 三、开发环境搭建

```bash
# Go
go version   # 1.21+
```

**安装步骤：**

```bash

cp config/config.example.yaml config/config.local.yaml
```

## 四、测试策略

### 4.1 单元测试

**测试覆盖率目标：>60%**

```go
// 示例：用户逻辑测试
func TestUserLogic_Register(t *testing.T) {
    // Arrange
    mockDB := &MockDB{}
    logic := NewUserLogic(mockDB)
    req := &RegisterRequest{
        Username: "testuser",
        Password: "123456",
    }

    // Act
    user, err := logic.Register(context.Background(), req)

    // Assert
    assert.NoError(t, err)
    assert.NotNil(t, user)
    assert.Equal(t, "testuser", user.Username)
}
```

**运行测试：**
```bash
# 运行所有测试
go test ./...

# 运行指定包测试
go test ./internal/logic/...

# 查看覆盖率
go test -cover ./...

# 生成覆盖率报告
go test -coverprofile=coverage.out ./...
go tool cover -html=coverage.out
```

---

### 4.2 接口测试

**使用Postman/Apifox**

1. 导入API集合

**示例测试脚本：**
```javascript
// Postman Tests
pm.test("Status code is 200", function () {
    pm.response.to.have.status(200);

pm.test("Response has data", function () {
    var jsonData = pm.response.json();

});
```

- [ ] 防火墙配置

**Nginx配置示例：**
```nginx
upstream backend {
    server 127.0.0.1:8080;
    keepalive 64;
}

server {
    listen 443 ssl http2;
    server_name yourdomain.com;

    ssl_certificate /path/to/cert.pem;
    ssl_certificate_key /path/to/key.pem;

    # 静态文件
    location /static/ {
        alias /var/www/static/;
        expires 7d;
    }

    # API接口
    location /api/ {
        proxy_pass http://backend;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
    }

    # WebSocket
    location /ws/ {
        proxy_pass http://backend;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
    }
}
```
