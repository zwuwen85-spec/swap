# 校园闲置物品交换平台 - 前端设计文档

## 一、前端技术选型

### 1.1 技术栈选择

| 技术 | 选型 | 说明 |
|-----|------|------|
| **框架** | Vue 3 | 渐进式框架，易学易用 |
| **构建工具** | Vite | 快速的前端构建工具 |
| **UI组件库** | Element Plus | 成熟的Vue3组件库 |
| **路由** | Vue Router 4 | 官方路由管理器 |
| **状态管理** | Pinia | Vue3官方推荐状态管理 |
| **HTTP客户端** | Axios | 易用的HTTP库 |
| **CSS预处理** | SCSS | CSS预处理器 |
| **代码规范** | ESLint + Prettier | 代码质量和格式化 |
| **多端方案** | uni-app | 可编译为H5/小程序 |

### 1.2 为什么选择这个技术栈

**Vue 3：**
- Composition API更灵活
- 性能更好（更小的包体积）
- 生态系统完善
- 学习曲线平缓

**Element Plus：**
- 组件丰富（60+组件）
- 文档完善
- 按需引入支持
- 主题定制方便

**uni-app：**
- 一套代码多端运行
- 支持H5、微信小程序、App
- 便于扩展移动端

## 五、状态管理

### 5.1 Pinia Store设计

#### User Store（用户状态）

```javascript
// store/user.js
import { defineStore } from 'pinia'
import { login, register, getUserInfo } from '@/api/user'

export const useUserStore = defineStore('user', {
  state: () => ({
    token: localStorage.getItem('token') || '',
    userInfo: null,
    isLogin: !!localStorage.getItem('token')
  }),

  getters: {
    userId: (state) => state.userInfo?.id,
    username: (state) => state.userInfo?.username,
    avatar: (state) => state.userInfo?.avatar
  },

  actions: {
    async login(params) {
      const res = await login(params)
      this.token = res.data.token
      this.userInfo = res.data
      this.isLogin = true
      localStorage.setItem('token', res.data.token)
    },

    async register(params) {
      const res = await register(params)
      this.token = res.data.token
      this.userInfo = res.data
      this.isLogin = true
    },

    async getUserInfo() {
      const res = await getUserInfo()
      this.userInfo = res.data
    },

    logout() {
      this.token = ''
      this.userInfo = null
      this.isLogin = false
      localStorage.removeItem('token')
    }
  }
})
```

```javascript
    path: '/admin',
    name: 'Admin',
    component: () => import('@/views/admin/Dashboard.vue'),
    meta: { title: '后台管理', requiresAuth: true, requiresAdmin: true }
  }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

// 路由守卫
router.beforeEach((to, from, next) => {
  const userStore = useUserStore()

  if (to.meta.requiresAuth && !userStore.isLogin) {
    next('/login')
  } else if (to.meta.requiresAdmin && userStore.userInfo?.role !== 'admin') {
    next('/')
  } else {
    next()
  }
})

export default router
```

```javascript
  timeout: 10000
})

// 请求拦截器
request.interceptors.request.use(
  (config) => {
    const userStore = useUserStore()
    if (userStore.token) {
      config.headers.Authorization = `Bearer ${userStore.token}`
    }
    return config
  },
  (error) => {
    return Promise.reject(error)
  }
)

// 响应拦截器
request.interceptors.response.use(
  (response) => {
    const res = response.data

    if (res.code !== 0) {
      ElMessage.error(res.message || 'Error')

      // Token失效
      if (res.code === 10002) {
        const userStore = useUserStore()
        userStore.logout()
        window.location.href = '/login'
      }

      return Promise.reject(new Error(res.message || 'Error'))
    }

    return res
  },
  (error) => {
    ElMessage.error(error.message || '请求失败')
    return Promise.reject(error)
  }
)

export default request
```
